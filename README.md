# Read_write_in_excel_using_Python

A Simple Python Project to use openpyxl and pandas lib to use it's features to extract , process and write data in excel.


Python Program :

1) Read data from Excel row wise,
2) Match keywords and maintain a counter(Rankings) for each keyword.
3) Whichever keyword appears the most , populate its associated category in the excel column called category.
4) Move to the next row and perform the steps 1-3 mentioned above, keep doing it tll it reaches EOF (Rows).
